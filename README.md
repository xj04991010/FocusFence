# 📊 FocusFence - 桌面整理與專注力工具

FocusFence 是一個專為 Windows 使用者打造的桌面生產力工具。它透過「分區收納 (Zones)」的概念，將凌亂的桌面轉化為井然有序的專注空間，並內建了番茄鐘與便利貼功能，協助您更高效地完成工作。

![FocusFence Header](https://raw.githubusercontent.com/xj04991010/FocusFence/main/FocusFence.ico)

## ✨ 核心功能

*   **📦 智慧分區收納 (Focus Zones)**
    *   在桌面上建立多個透明美觀的「框框」，將檔案按專案或類型分類。
    *   支援拖曳收納，自動將檔案移動至對應的子資料夾，保持桌面整潔。
*   **⏲️ 內建番茄鐘 (Pomodoro Timer)**
    *   整合在每個收納分區中的番茄鐘，幫助您追蹤每個任務的投入時間。
    *   支援自定義專注與休息時間，並提供週目標進度追蹤。
*   **📝 即時便利貼 (Sticky Notes)**
    *   每個分區都有獨立的便利貼空間，隨手記錄待辦事項或專案備註。
*   **📂 高效檔案管理**
    *   支援在框框內直接「左鍵雙擊」進入子資料夾。
    *   右鍵選單整合「📦 解壓縮」功能，無需開啟檔案總管即可處理壓縮檔。
*   **🎨 現代感介面設計**
    *   採用磨砂玻璃感 (Glassmorphism) 與動態配色設計。
    *   支援沉浸模式 (Smart Dormancy)，在您不需要時自動收縮，減少視覺干擾。

## 🚀 快速上手

1.  **下載與安裝**：從 [Releases](https://github.com/xj04991010/FocusFence/releases) 下載最新的安裝包。
2.  **建立分區**：啟動後，使用預設的「Inbox」、「Active」、「Arsenal」分區，或在控制台中建立新分區。
3.  **收納檔案**：將桌面的檔案直接拖入框框，它們會被自動移動到對應的資料夾中。
4.  **開始專注**：點擊框框標題開啟番茄鐘，開始您的深度工作。
